# Personal Portfolio Website

## Setup

1. `cd server && cp .env.example .env`
2. Add your MongoDB Atlas URI and frontend URL.
3. Install dependencies in both `client` and `server`.
4. Run backend with `npm run dev`.
5. Run frontend with `npm run dev`.

## Deploy

- Deploy backend to Render.
- Deploy frontend to Netlify.
- Set `VITE_API_URL` in Netlify environment variables.
